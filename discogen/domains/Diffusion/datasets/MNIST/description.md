DESCRIPTION
MNIST is a classic dataset of handwritten digits, consisting of grayscale images of digits 0-9. It serves as a fundamental benchmark for image classification algorithms, providing a standardized testbed for evaluating machine learning models on a relatively simple visual recognition task with minimal computational requirements.


OBSERVATION SPACE
Each observation is a 28x28 grayscale image with pixel values in the range 0, 255, where higher values represent darker pixels. Total of 784 pixels per image.


CLASSES
The dataset contains 10 digit categories:

0: Zero
1: One
2: Two
3: Three
4: Four
5: Five
6: Six
7: Seven
8: Eight
9: Nine


DATASET STRUCTURE
Training set: 60,000 images (6,000 per class)
Test set: 10,000 images (1,000 per class)
