DATASET OVERVIEW
Training set: 19,850 images (50 per class) Test set: 19,850 images (50 per class) The training and test data are divided into approximately 10 partitions for cross-validation, with standard splits provided for evaluation.


DESCRIPTION
SUN397 is a large-scale scene understanding database designed for developing and benchmarking scene recognition and classification systems. It contains 397 well-sampled scene categories, with each concept (scene category) represented by multiple images covering diverse conditions and viewpoints.


OBSERVATION
Each observation is a full-color (RGB) image of varying resolution. In practice, images are often resized (e.g., to 224×224 or 256×256) during preprocessing for deep learning models.


DATASET STRUCTURE

Training set: ~19,850 images Validation set: ~19,850 images Test set: ~19,850 images
Total: ~108,754 images across 397 scene categories
